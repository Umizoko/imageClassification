# Image Classification 

画像を分類するwebサイト

![image](screenshot/screenshot.png)

## Description

MLライブラリを使用したWebサイトです。
選択した画像を事前に学習したモデルで分類し、予測スコア
の高い順に結果を表示します。

website: [https://umizoko.github.io/imageClassification/](https://umizoko.github.io/imageClassification/)

## Requirement
- ml5.js
- jquery
- highcharts
